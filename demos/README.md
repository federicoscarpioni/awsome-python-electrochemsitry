# Demos

Worked examples for the libraries listed in the main [README](../README.md), showing each library's main functionality and, where relevant, comparisons between libraries that cover the same purpose.

## Layout

Notebooks are grouped by category, matching the sections of the main README:

```
demos/
  eis/
    pyimpspec_demo.ipynb
```

## Conventions

- One notebook per library (`<library>_demo.ipynb`), executed with outputs saved so it renders directly on GitHub.
- The first code cell installs the library with `%pip install -q ...`, so the notebook is runnable standalone without a separate requirements file.
- Where it doesn't require a real instrument file, demos use synthetic/mock data generated by the library itself, so they run without external dependencies.
